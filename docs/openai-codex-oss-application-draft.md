## Project Description

Chinese:

```text
轉角遇到貓 是一个以开源核心为基础、手机优先的猫咪相遇地图和贴纸图鉴。它帮助用户记录在街角、社区、旅途中遇到的猫咪，包括照片、名字、备注和地点线索，并可以用隐私友好的方式分享猫咪卡片、猫咪地图海报和地图数据。

目前开源的核心是一个 local-first 的 React / TypeScript PWA。未来的云端同步、公开共享地图、进阶 AI 功能、储存空间、审核系统和订阅功能，可能会作为独立的商业服务开发；但本地优先的核心应用会保持开放、可学习、可检查。
```

English:

```text
FoundCat, known in Chinese as 轉角遇到貓, is an open-source-core, mobile-first cat encounter map and sticker archive. It helps people document neighborhood cats with photos, names, notes, and location context, then share privacy-aware cat cards, cat map posters, and exportable map data.

The open-source core is a local-first React/TypeScript PWA. Future hosted services such as cloud sync, public shared maps, premium AI workflows, storage, moderation, and subscription features may be developed separately while keeping the local-first core useful and inspectable as open source.
```
